Voxly – Facebook User Data Deletion

If you have authorized the Voxly app via Facebook and wish to delete your data, you can follow the instructions below.

Manual Deletion Request

You can request deletion of your user data by emailing us at:

support@voxly.app

Please include your Facebook User ID or Instagram Business ID in the message, if available.

Automated Deletion Callback (for Facebook)

If you’re attempting to remove the app via Facebook and invoke data deletion automatically, our app is configured to accept Facebook’s data deletion request callback.

For Facebook to trigger automatic data deletion, visit:

https://www.facebook.com/settings?tab=applications

Response Format

When Facebook sends a data deletion request, our system responds with a JSON payload like this:

{
  "url": "https://voxly.app/data-deletion-status?request_id=",
  "confirmation_code": ""
}

For more information, see the Facebook Data Deletion Documentation .
